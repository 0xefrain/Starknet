## Requirements

Before you begin, you need to install the following tools:

```bash
  npm 
```


## Quickstart

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd 
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
